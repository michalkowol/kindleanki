# Kindle to Anki

Kindle to Anki exporter

## Installation

```bash
brew install poetry
```

```bash
poetry install
```

## Usage

```bash
poetry run app --deepl-api-key YOUR_DEEPL_API_KEY --input /Volumes/Kindle/system/vocabulary/vocab.db --output /Users/michal/Dropbox/referencje/kindle/vocab.csv
```
